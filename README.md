# 💼 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, and contact information in a clean and user-friendly interface.

---

## 🚀 Features

- 🌐 Fully responsive design (mobile, tablet, desktop)
- 🌙 Dark / Light mode toggle with localStorage support
- 📱 Mobile navigation with hamburger menu
- 🎯 Smooth scrolling navigation
- ⬆️ Scroll-to-top button
- 💡 Clean UI with modern layout (Flexbox & Grid)
- 📬 Contact form (frontend demo)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid + Custom Properties)
- JavaScript (Vanilla JS)
- Font Awesome (icons)
- Google Fonts (Inter)
